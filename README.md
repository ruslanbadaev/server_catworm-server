# vpsWorm

#### Clone:
```
git clone https://github.com/ruslanbadaev/vpsWorm.git
```

#### Install modules:
```
cd vpsWorm
npm i
```

#### Generate QR for mobile app and kill last session:
```
npm run gen
```

#### Generate QR of current session:
```
npm run show
```

#### Run app:
```
npm start
```

#### Stop app:
```
npm stop
```

#### Restart app:
```
npm restart
```

... and write the secret word to generate qr
